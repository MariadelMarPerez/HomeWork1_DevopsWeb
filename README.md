# *Guía de Documentación del Proyecto*

Este archivo contiene la información fundamental para entender, instalar y utilizar el repositorio de manera correcta.

---

## -*Propósito del Proyecto*
El objetivo principal de este desarrollo es crear una tarjeta de presentación profesional interactiva ,ademas aplicar conocimientos  basicos de **HTML5**, **CSS3** y **JavaScript**

## -**Instrucciones de Instalación**
---
Para tener este proyecto funcionando en tu computadora local, sigue estos pasos:

1. **Clonar el repositorio:**
   Utiliza el comando de Git en tu terminal:
   ```bash
   git clone [https://github.com/MariadelMarPerez/HomeWork1_DevopsWeb.git] (https://github.com/MariadelMarPerez/HomeWork1_DevopsWeb.git)

2. **Entrar a la carpeta:**
>cd: HomeWork1_DevopsWeb
---
Asegúrate que existan los archivos index.html, style.css, script.js y la carpeta /src con las imágenes necesarias.

3. **Abrir el proyecto:**
> Simplemente abre el archivo index.html en tu navegador o usa la extensión Live Server en Visual Studio Code.

## **-Instrucciones de Uso**
>Una vez instalado, puedes interactuar con el proyecto de la siguiente forma:

**Visualización**: Observa el perfil profesional y la foto cargada.

**Interactividad**: Haz clic en el botón de "Otras opciones" para ver los mensajes dinámicos generados por JavaScript.

Navegación: Utiliza el enlace directo a LinkedIn para conectar con la autora

## **-Importante**
>**Configuración de Imágenes**: Asegúrate de que la carpeta src contenga los archivos **image.jpg y fondo4.jpg.** Sin estos nombres exactos, el diseño no se mostrará correctamente.
